Hexo - Windows use

1 安装Hexo

  • 安装Node.js

  • 安装Git

  • 检查安装是否成功,打开git bsah ，输入命令如下，显示正确的版本号即成功安装。 

     node -v    
 npm -v

  • 使用npm安装hexo并检查版本–创建一个blog文件夹用git bash打开进行hexo安装

      npm install -g hexo-cli
  hexo -v

  • hexo初始化–在安装的目录下，进行初始化。folder为博客需要的所有文件名（自定义比如就叫myblog）

      hexo init <folder>  --初始化，生成hexo需要的文件
  cd <folder>  		--进入文件
  npm install  		--工具安装
